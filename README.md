Esta segunda parte del proyecto profundiza en el **análisis de evasión de clientes (Churn)** en una empresa de telecomunicaciones. A partir de un dataset de **7,043 clientes**, se exploraron patrones relacionados con el comportamiento de consumo, antigüedad, contratos y servicios utilizados para comprender mejor los factores asociados al abandono del servicio.

El análisis utiliza **visualizaciones, correlaciones y herramientas de machine learning** para identificar tendencias relevantes que puedan ayudar a diseñar estrategias de **retención de clientes**.

---

## 🎯 Objetivo

Profundizar en los patrones de evasión de clientes mediante **análisis exploratorio de datos (EDA)** y técnicas de **machine learning**, identificando variables clave que influyen en la probabilidad de churn.

---

## 🔎 Aspectos analizados

Durante el análisis se evaluaron múltiples variables del dataset:

- Antigüedad del cliente (**Tenure**)
- Cargos mensuales y gasto del cliente
- Tipo de contrato
- Tipo de servicio de internet
- Variables demográficas
- Relación entre número de servicios y costo mensual
- Correlaciones entre variables

---

## 📈 Principales hallazgos

El análisis permitió identificar varios factores asociados al churn:

- Los **clientes con menor antigüedad** presentan mayor probabilidad de abandono.
- Los **contratos mes a mes** concentran la mayoría de las cancelaciones.
- Los **cargos mensuales elevados** están asociados a mayor tasa de churn.
- El servicio de **fibra óptica** presenta una proporción importante de cancelaciones.
- Los clientes con **pareja o dependientes** muestran mayor estabilidad.

---

## 💡 Recomendaciones

A partir de los resultados obtenidos se proponen algunas estrategias para reducir la evasión de clientes:

- Incentivar **contratos de mayor duración** mediante beneficios o descuentos.
- Implementar **programas de retención temprana** para clientes nuevos.
- Revisar la **estrategia de precios del servicio de fibra óptica**.
- Diseñar **paquetes de servicios (bundles)** que aumenten el valor percibido.
- Promover **métodos de pago automáticos** para reducir cancelaciones.
